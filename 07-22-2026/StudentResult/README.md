## Introduction to JavaScript, Variables, Data Types & Operators

### What I Learned

* **JavaScript Basics**: Learned how to connect a JavaScript file to an HTML page and execute code.
* **Variables**: Used `let` and `const` to store student information and marks.
* **Data Types**: Worked with strings for names and numbers for marks.
* **Operators**: Used arithmetic operators (`+`, `/`) to calculate the total and average, and comparison operators (`>=`) to determine the student's grade.
* **DOM Manipulation**: Retrieved input values from HTML and displayed the calculated result on the webpage.

### My Task

Built a **Student Grade Calculator**. The user enters a student's name, class, and marks for five subjects. The application calculates the total, average, assigns a grade, and displays a remark based on the student's performance.
