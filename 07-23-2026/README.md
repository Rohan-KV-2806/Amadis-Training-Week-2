# FAN Media

A mini social media app built with HTML, CSS, and JavaScript.

## What it does

- Sign up & log in (localStorage-based auth)
- Create text posts with date/time stamp
- View your posts on a dashboard
- Like posts (count persists in localStorage)

## Pages

| Page | Route |
|------|-------|
| Signup | `signup.html` |
| Login | `login.html` |
| Dashboard | `index.html` |
| Create Post | `post.html` |

## Directory Structure

FAN Media/

├── index.html # Dashboard — view & like posts

├── login.html # Login form

├── signup.html # Signup form

├── post.html # Create a new post

├── styles/

│ ├── auth.css # Shared auth/post page styles

│ └── dashboard.css # Dashboard-specific styles

└── scripts/

├── auth.js # Signup & login logic

├── dashboard.js # Post feed & likes

└── post.js # Post creation & storage
