# FAN Media

A mini social media app built with HTML, CSS, and JavaScript.

## What it does

- Sign up & log in (localStorage-based auth)
- Create text posts with date/time stamp
- View your posts on a dashboard
- Like posts (count persists in localStorage)

## Pages

| Page | Route |
|------|-------|
| Signup | `signup.html` |
| Login | `login.html` |
| Dashboard | `index.html` |
| Create Post | `post.html` |
