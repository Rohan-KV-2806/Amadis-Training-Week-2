# FAN Media

A mini social media app built with HTML, CSS, and JavaScript.

## What it does

- Sign up & log in (localStorage-based auth)
- Create text posts with date/time stamp
- View your posts on a dashboard
- Like posts (count persists in localStorage)

## Pages

| Page | Route |
|------|-------|
| Signup | `signup.html` |
| Login | `login.html` |
| Dashboard | `index.html` |
| Create Post | `post.html` |
## 📁 Directory Structure

```text
FAN Media/
├── index.html              # Dashboard - View and like posts
├── login.html              # User login
├── signup.html             # User registration
├── post.html               # Create a new post
│
├── styles/
│   ├── auth.css            # Styles for login, signup & post pages
│   └── dashboard.css       # Dashboard styles
│
└── scripts/
    ├── auth.js             # Signup, login & logout logic
    ├── dashboard.js        # Load posts & handle likes
    └── post.js             # Create and save posts
```
> **CSS is added by AI**
